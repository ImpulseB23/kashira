# Kashira

A modular desktop toolkit. Install the blades you need, disable what you don't.

> **Early development** - not ready for daily use yet.

## Blades

- **Timeshift** - Know when someone's likely to respond based on their timezone, age, and schedule

More blades coming soon.

## Install

Grab the latest build from [Releases](https://github.com/ImpulseB23/kashira/releases).

## Build from source

Requires [Bun](https://bun.sh), [Rust](https://rustup.rs), and a C++ compiler.

```
git clone https://github.com/ImpulseB23/kashira.git
cd kashira/habaki
bun install
cargo tauri dev
```
